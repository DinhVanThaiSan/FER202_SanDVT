# Lab 5: About React Router

Online Quiz Application using React Router, React Bootstrap and Bootstrap.

## Pages

- Home: display slider and menu images
- About: display project information
- News: display news list from an array
- Quiz: display questions and answer options
- Contact: display contact form using React Bootstrap

## How to run

```bash
npm install
npm start
```

Open the URL shown in the terminal, usually:

```text
http://localhost:5173
```
